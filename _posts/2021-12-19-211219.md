### 제네릭이란?
하나의 인터페이스를 선언해서 여러가지 유형의 타입으로 쓸 수 있다.
```
interface User<T> {
  name: string;
  phone: T;
}

const user1:User<number> = {
  name: 'test1',
  phone: 123
}

const user2:User<string> = {
  name: 'test2',
  phone: '123'
}
```

### 유틸리티타입
- keyof: 모든 키 값을 키로 만들어준다.
- Prticial : 모든 옵션을 옵셔널하게만들어준다.
- Require: 모든 옵션을 필수로 만들어준다.
-> 새로운 값을 추가할 수는 없다.
- Pick: 타입에서 특정 속성만 사용할 수 있다.
- Omit: 타입에서 해당 키값을 제외하고 사용할 수 있다.
